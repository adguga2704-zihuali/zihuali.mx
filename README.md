# zihuali.mx
Propósito: Empoderar a las mujeres nahuas de la Sierra Norte de Puebla, transformando su herencia cultural en joyería de diseño contemporáneo bajo un modelo de comercio justo. 2. Definición Estratégica  * Misión: Crear un puente entre el arte ancestral y el mundo moderno, dignificando el trabajo de la mujer indígena y ofreciendo piezas con alma.
